[employees table]

a. Write a query to select all columns and rows from the employees table:
Answer: SELECT * FROM employees;

Explanation: This command retrieves all columns and rows from the employees table, returning a result set of all employees' information.

b. Write a query to select only the name and salary columns of all employees with a salary greater than 50000:
Answer: SELECT name, salary FROM employees WHERE salary > 50000;

Explanation: This command retrieves only the name and salary columns of all employees whose salary is greater than 50000 from the employees table.

c. Write a query to calculate the average salary of all employees:
Answer: SELECT AVG(salary) FROM employees;

Explanation: This command calculates the average salary of all employees from the employees table and returns a single value result.

d. Write a query to count the number of employees who work in the "Marketing" department:
Answer: SELECT COUNT(*) FROM employees WHERE department_id = (SELECT id FROM departments WHERE name = 'Marketing');

Explanation: This command retrieves the id of the "Marketing" department from the departments table and uses it to count the number of employees who work in that department from the employees table.

e. Write a query to update the salary column of the employee with an id of 1001 to 60000:
Answer: UPDATE employees SET salary = 60000 WHERE id = 1001;

Explanation: This command updates the salary column of the employee with an id of 1001 to 60000 in the employees table.

f. Write a query to delete all employees whose salary is less than 30000 :
Answer: DELETE FROM employees WHERE salary < 30000;

Explanation: This command deletes all employees whose salary is less than 30000 from the employees table.

[departments table]

a. Write a query to select all columns and rows from the departments table:
Answer: SELECT * FROM departments;

Explanation: This command retrieves all columns and rows from the departments table, returning a result set of all departments' information.

b. Write a query to select only the name and manager columns of the "Finance" department:
Answer: SELECT name, manager FROM departments WHERE name = 'Finance';

Explanation: This command retrieves only the name and manager columns of the "Finance" department from the departments table.

c. Write a query to calculate the total number of employees in each department:
Answer: SELECT departments.name, COUNT(*) FROM employees JOIN departments ON employees.department_id = departments.id GROUP BY departments.name;

Explanation: This command joins the employees and departments tables on the department_id and id columns, respectively, and calculates the total number of employees in each department by grouping them by the department name.

d. Write a query to insert a new department called "Research" with a manager named "John Doe":
Answer: INSERT INTO departments (name, manager) VALUES ('Research', 'John Doe');

Explanation: This command inserts a new row into the departments table with the name "Research" and the manager "John Doe".
